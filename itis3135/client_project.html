Client Project

Project Overview

    The development of this application aims to inform and engage users with the experiences of Mach Atlus's aviation history and services. It provides users the opportunity to meet the faces behind the company, learn about its background in the industry, and request services based on aircraft needs by form submission. The targeted audience is intended for the following:
    • Aircraft Owners and Operators
    • Aircraft Inspection & Maintenance Professionals
    • Aircraft Maintenance Corporations

Dynamic Functionality

  • Home Page
    • The dynamic functionality of the home page's search box will allow users to search the contents of all pages of the application.
  • Services
    • The dynamic functionality of this page will include functions that validate if the form is missing entries (via an alert), explain (via alerts) what a field is asking, and alert the user that their request has been sent.
  • Contact Us
    • The dynamic functionality of this page will include functions that validate if the form is empty and when the form has been submitted (via alerts).
  • Gallery
    • The dynamic functionality of this page provides users with interactivity for image browsing through an image rotator.
